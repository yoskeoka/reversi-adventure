# ADR Template

Based on the [MADR (Markdown Any Decision Records)](https://adr.github.io) format.
Copy the template below when adding a new record to [adr.md](adr.md).

---

```markdown
## [YYYY-MM-DD] Title of Decision

**Status:** proposed | accepted | deprecated | superseded by [ADR-YYYY-MM-DD]

### Context and Problem Statement

Describe the context and problem statement in two to three sentences.

### Decision Drivers

- [Driver 1]
- [Driver 2]

### Considered Options

1. [Option 1]
2. [Option 2]
3. [Option 3]

### Decision Outcome

Chosen option: [Option N], because [justification].

### Pros and Cons of the Options

#### [Option 1]

- Good, because [argument]
- Bad, because [argument]

#### [Option 2]

- Good, because [argument]
- Bad, because [argument]
- Rejected because [reason]

### Consequences

[Positive, negative consequences, and mitigations.]
```
